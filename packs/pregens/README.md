# Pregenerated Heroes Compendium

This compendium contains pregenerated hero characters for the Singularity system.

## How to Import Pregens

To import pregenerated characters from the handbook:

1. Open Foundry VTT
2. Open the console (F12)
3. Run the import script (see module/utilities/import-pregens.js)
4. Or use the macro provided in the system

## Available Pregens

- **Vanguard**: A tanky hero with the Bastion powerset, specializing in defense and endurance.
