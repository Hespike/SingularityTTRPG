# Enemies Compendium

This compendium contains enemy actors for the Singularity system.

## How to Add Enemies

To add enemies to this compendium:

1. Open Foundry VTT
2. Go to the Compendium Packs tab
3. Find the "Enemies" compendium
4. Click the "+" button to create a new enemy actor
5. Fill in the enemy's details based on the examples below

## Example Enemies

### Thug (CR 1/2)
- **HP**: 8
- **AC**: 12 (11 Jacket + 1 AGI)
- **Speed**: 25 ft.
- **Abilities**: MGT +1, AGI +1, END 0, WIT 0, CHA 0
- **Weapons**: Combat Knife (1d20+1, 1d4+1 kinetic)
- **Skills**: Apprentice with Intimidation (Charm)
- **Traits**: Living
- **Equipment**: Combat Knife, Jacket

### Small Blade Drone (CR 1/2)
- **HP**: 6
- **AC**: 13 (11 Natural armor + 2 AGI)
- **Speed**: 30 ft. (Fly)
- **Abilities**: MGT +1, AGI +2, END 0, WIT 0, CHA 0
- **Weapons**: Retractable Blade (1d20+1, 1d4+1 kinetic)
- **Traits**: Electrical, Non-living

### Soldier (CR 1)
- **HP**: 15
- **AC**: 17 (14 Vest + 3 AGI)
- **Speed**: 25 ft.
- **Abilities**: MGT +2, AGI +3, END +2, WIT 0, CHA 0
- **Weapons**: Rifle (1d20+7, 1d6+3 kinetic)

- **Special Abilities**: Trained with Medium Armor
- **Traits**: Living
- **Equipment**: Rifle, Combat Vest

### Cyber-Enhanced Enforcer (CR 5)
- **HP**: 45
- **AC**: 18 (18 Plated Assault Armor)
- **Speed**: 25 ft.
- **Abilities**: MGT +3, AGI +2, END +3, WIT +1, CHA 0
- **Weapons**: Greatsword (1d20+11, 1d12+3 photonic)
- **Skills**: Competent with Heavy Melee Weapons, Competent with Unarmed Attacks, Competent with Endurance Saves
- **Special Abilities**: Trained with Heavy Armor, Cyber-Enhanced Endurance
- **Traits**: Electrical, Living
- **Weakness**: Lightning 3
- **Equipment**: Greatsword (with Damage Type Conversion - Photonic), Plated Assault Armor

## Notes

- Challenge Rating (CR) indicates the difficulty level
- AC is calculated as 10 + Agility modifier + Armor bonus
- HP values shown are maximum values
- Ability scores are shown as modifiers (e.g., +1 means the ability score is 12-13)
